Elementos de una página web

HTML

Es un lenguaje de marcado que nos permite indicar la estructura de un documnto web mediante etiquetas

CSS

Hojas de Estilo en Cascada (del inglés Cascading Style Sheets) es el lenguaje de estilos utilizado para describir la presentación de documentos web

JavaScript

JavaScript es un lenguaje de programación diseñado en un principio para añadir interactividad a las páginas web.
