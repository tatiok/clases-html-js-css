Global Attributes

Attribute Description
class Specifies one or more class names for an element.
hidden It specifies that an element is not yet, or is no longer, relevant.
id This attribute specifies a unique id for an HTML element (the value must be unique within the HTML document).
name The name attribute specifies a name for an HTML element.
style This attribute specifies an inline style for an element.
title The title attribute specifies extra information about an element. The information is most often shown as a tooltip text when the mouse moves over the element.
