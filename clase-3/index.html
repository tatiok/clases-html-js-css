Most Common HTML Tags


Tag Description
<!--...--> The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers.
<!DOCTYPE> All HTML documents must start with a declaration.The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.
<a> This tag defines a hyperlink, which is used to link from one page to another.
<b> The tag specifies bold text without any extra importance.
<body> The tag defines the document's body.The <body> element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
<br> The tag inserts a single line break.
<button> The tag defines a clickable button. Inside a <button> element you can put text (and tags like <i>, <b>, <strong>, <br>, <img>, etc.). That is not possible with a button created with the <input> element.
<canvas> The tag is used to draw graphics, on the fly, via scripting (usually JavaScript).
<div> The tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.
<form> The tag is used to create an HTML form for user input.
<h1> to <h6> The <h1> to <h6> tags are used to define HTML headings.
<head> The element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.
<hr> The element is most often displayed as a horizontal rule that is used to separate content (or define a change) in an HTML page.
<html> The tag represents the root of an HTML document.
<i> The tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.
<iframe> The tag specifies an inline frame. An inline frame is used to embed another document within the current HTML document..
<img> The tag is used to embed an image in an HTML page.
<input> The tag specifies an input field where the user can enter data.
<label> The tag defines a label for several elements
<li> The tag defines a list item
<link> The tag defines the relationship between the current document and an external resource
<map> The tag is used to define an image map. An image map is an image with clickable areas
<meta> The tag is defines metadata about an HTML document. Metadata is data (information) about data
<ol> The tag is defines an ordered list. An ordered list can be numerical or alphabetical.
<optgroup> The tag is used to group related options in a <select> element (drop-down list).
<option> The tag defines an option in a select list.
<p> The tag defines a paragraph.
<pre> The tag defines preformatted text
<script> The tag is used to embed a client-side script (JavaScript)
<select> The tag is used to create a drop-down list.
<span> is an inline container used to mark up a part of a text, or a part of a document.
<strong> The tag is used to define text with strong importance. The content inside is typically displayed in bold
<style> The tag is used to define style information (CSS) for a document.
<svg> The tag has several methods for drawing paths, boxes, circles, text, and graphic images.
<table> The tag defines an HTML table.
<textarea>>> The tag defines a multi-line text input control.
<thead> The tag is used to group header content in an HTML table.
<tbody> The tag is used to group the body content in an HTML table.
<tfood> The tag is used to group footer content in an HTML table.
<tr> The tag defines a row in an HTML table.
<th> The tag defines a header cell in an HTML table.
<td> The tag defines a standard data cell in an HTML table.
<title> The tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.
<u> The tag represents some text that is unarticulated and styled differently from normal text, such as misspelled words or proper names in Chinese text. The content inside is typically displayed with an underline. You can change this with CSS
<ul> The tag defines an unordered (bulleted) list.
<video> The tag is used to embed video content in a document, such as a movie clip or other video streams.
